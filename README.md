Morphverse 是一款反抗“树状文件夹体系”的非线性个人知识管理与灵感碰撞引擎。它基于**德勒兹的“根茎 (Rhizome)”理论**与**数学范畴论 (Category Theory)** 构建，专为拥有旺盛好奇心、极度发散的高维思考者打造。

在这里，知识不再是被隔离在抽屉里的死板文件，而是可以无限蔓延、自动复合的生命体。

---

## 💥 The Problem: 传统笔记的“认知瘫痪”

在使用传统双链笔记（如 Notion, Obsidian）时，我们往往会陷入绝望的“收纳师陷阱”：
- 每次闪现一个绝妙的灵感，系统都强迫你思考：“它属于哪个文件夹？该打什么标签？”
- 这种极高的认知阻力，导致灵感“记下来就等于扔进垃圾桶”。
- 记录的知识点互不相干，无法产生“跨界同构”的化学反应。

## 🪐 The Philosophy: 万物皆态射 (Everything is a Morphism)

Morphverse 彻底抛弃了静态笔记的隐喻。在这里，每一次认知升级都被定义为一个**态射 (Morphism)**——即知识从起点向终点的动态演化。

- **Domain (起点 / 输入):** 遇到的痛点、未解的难题、或前置条件。
- **Codomain (终点 / 输出):** 发现的解法、契约、或推导出的结论。
- **Adjoint Functors (伴随函子双切面):** 所有的输入/输出都具备两个维度。你可以用大白话记录它的“应用范畴（解决什么现实问题）”，也可以一键切换到“原理范畴（底层的严谨定义）”。

## ✨ Core Features: 杀手级特性

### 1. 零阻力捕获与显性连接
抛弃分类。只需填入起点和终点，系统即刻生成一条态射。手动将它与库中任意一个看似遥远的节点相连，极大降低未来检索信息的认知负荷。

### 2. 复合灵感库
这是 Morphverse 的灵魂 Aha Moment。
自然语言无法像数学公式那样严谨推导。因此，后台通过 LLM 与向量检索，每天自动计算库中毫无关联概念的“语义同构性”。
- 系统会向你推送意想不到的“模糊复合连接”（例如：将物理定律与智能合约强行复合）。
- 这种因信息衰减而产生的离谱结论，在这里不仅不是 Bug，反而是一种达达主义式的诗意。这是德勒兹的“逃逸线”，是你打破思维定式的顶级灵感源泉。

### 3. 多巴胺热力引擎
告别强迫症式的打卡。参考 GitHub 的热力图日历，你每天创建的态射越多，当天的方块颜色越深。用最直观的拓扑视觉快感，驱动你不断探索新大陆。
