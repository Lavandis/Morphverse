# 🌌 Morphverse - 核心架构与功能蓝图

> **产品共识**：
> Morphverse 是一款反抗“树状文件夹”的非线性思考引擎。核心机制在于“态射复合（知识碰撞）”。
> **UI 原则**：极简、克制。对标 Notion 和 Obsidian 的极简美学（主色调为紫色）。拒绝任何多余的视觉干扰，让注意力 100% 聚焦于知识的流向与逻辑。

## 🛠 一、 技术栈设定
* **前端框架**：`Vite` + `React` + `TypeScript`
* **样式方案**：`Tailwind CSS`
* **状态管理**：`Zustand` 或 `React Context`（早期原型数据存放于本地内存）
* **UI 图标**：`Lucide React`（极简线条风格）

## 🎯 二、 核心功能模块 (三大核心流)
系统由三个极其精简的模块构成闭环：**捕获 ➔ 沉淀 ➔ 碰撞**。

### 模块 1：极简态射录入台 (Quick Capture)
* **功能定位**：以最低的认知阻力，捕获脑海中闪现的灵感与逻辑链条。
* **UI 呈现**：页面顶部或侧边栏固定的极简表单，无复杂富文本选项。
* **核心字段**：
    * **概念名 (Title)**：知识点的主题。
    * **输入 (Input)**：前置条件 / 存在痛点 / 待解决的问题。
    * **输出 (Output)**：推导结论 / 对应解法 / 最终结果。
    * **标签 (Tags)**：（可选）用于基础的领域分类。
* **💡 运作实例**：
    * *概念名*：番茄工作法
    * *输入*：极易分散的注意力与长线任务带来的畏难情绪。
    * *输出*：通过 25 分钟专注与 5 分钟休息的物理切割，维持多巴胺稳定释放。
* **交互**：录入完毕后回车，表单瞬间清空，数据无缝滑入下方的知识库。

### 模块 2：线性知识库 (Morphism Feed)
* **功能定位**：按时间轴沉淀所有的思维碎片，清晰展示所有的映射关系。
* **UI 呈现**：极其干净的无序列表结构。
* **详细说明**：
    * 数据按时间倒序排列。
    * 每张卡片清晰标明逻辑流向：`[输入] ➔ [输出]`。
    * 附带极简的文本搜索过滤功能。
    * 每张卡片提供一个 **复选框 (Checkbox)**，用于触发下游的“复合”动作。

### 模块 3：复合工作台 (Composition Arena) —— 核心灵魂
* **功能定位**：系统将两个看似遥远、毫无关联的知识点强行连接，诱发跨界顿悟。
* **触发条件**：用户在知识库列表中，精准勾选了任意 **2 个** 态射卡片。
* **UI 呈现与交互**：
    1. 屏幕划分出专属的工作台区域，两张被选中的卡片并列展示。
    2. 中间浮现出一个醒目的核心操作按钮：**`[ 执行复合 (Compose) ]`**。
    3. 点击该按钮，系统模拟 AI 推理过程（短暂加载），随后在下方以**打字机特效**逐字生成一段全新的“复合洞见”。
* **💡 复合推演实例**：
    * *态射 A*：【专注力模型】输入：多巴胺分泌不足 ➔ 输出：依赖高压刺激产生极度专注。
    * *态射 B*：【RAG 检索增强】输入：大模型易产生幻觉 ➔ 输出：外挂本地知识库强制基于事实推理。
    * *生成结果*：
        > **[跨界洞见 - 置信度 85%]**：将容易涣散的大脑视为缺乏长期记忆的算力节点。其应对复杂任务的最佳解法，是将外部的极简知识库（如 Morphverse）作为物理外挂。用外界的即时线索（RAG 检索）去定向触发大脑短暂但极具爆发力的脉冲式算力。
